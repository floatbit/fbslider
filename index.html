fbslider

About

A jQuery plugin to be used with jQuery Mobile. Virtual mouse events (vclick, vmousemove, etc) are bound to the slider controls. Multiple sliders can be present on a single page.

Requirements

  • jQuery 1.5.2
  • jQuery Mobile Alpha 4.1

Instructions

HTML

The number of div.set must match the number of li.dot. You can put whatever you like in the sets. 

<div class="fbslider">
  <div class="sets">
    <div class="set"><img src="river1.jpg" width="296" height="200"/></div>
    <div class="set"><img src="river2.jpg" width="296" height="200"/></div>
    <div class="set"><img src="river3.jpg" width="296" height="200"/></div>
    <div class="set"><img src="river4.jpg" width="296" height="200"/></div>
   </div>
  <div class="control">
    <ul>
      <li class="cap-start">&nbsp;</li>
      <li class="dot">&bull;</li>
      <li class="dot">&bull;</li>
      <li class="dot">&bull;</li>
      <li class="dot">&bull;</li>
      <li class="cap-end">&nbsp;</li>
     </ul>
    <div class="handle">&bull;</div>
  </div>
</div>

JavaScript

$('.fbslider').fbslider('init');

CSS

Set styles of controls, set widths, etc. One thing - the handle width must match the width of li.dot
