Slider

About

A jQuery plugin to be used with jQuery Mobile. Virtual mouse events (vclick, vmousemove, etc) are bound to the slider controls. Calculations such as masking and movement amounts are calculated in fbslider.js and initial values such as slider width, and sets to move are set in css. Multiple sliders can be present on a single page.

Requirements

  • jQuery 1.5.2
  • jQuery Mobile Alpha 4.1

    <ul>

</div>
